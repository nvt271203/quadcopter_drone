Quadcopter Package Delivery, Weather Conditions

This example shows how weather conditions can be obtained from the online weather service OpenWeather. A Python script is called from within MATLAB to obtain weather for a specific location, either current conditions or a forecast for the next 5 days.

The Python module reads and parses current weather data from the web API: https://openweathermap.org/api
